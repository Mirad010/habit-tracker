# MP2 Proposal — Study Timer

## The Problem
When I study, I lose track of time and either study too long without breaks or forget to start studying at all. I need a simple timer to keep me focused.

## The Solution
A study timer web app that lets me set a countdown, see the time ticking, and get an alert when time is up.

## Features
- Set custom study time (e.g. 25 minutes)
- Start, pause, and reset the timer
- Alert when time runs out
- Simple and clean design

## Who is it for
For myself and any student who needs help staying focused while studying.
