# Habit Tracker Pro

A professional daily habit tracking tool built with vanilla JavaScript.

## Live Site
[Open Habit Tracker](https://mirad010.github.io/study-timer)

## What it does
- Add habits with categories (Health, Study, Personal)
- Mark habits complete each day
- Streak counter with best streak memory
- Progress bar toward 30-day mastery
- Data persists in browser with localStorage

## How to use
1. Type a habit name and select a category
2. Click Add
3. Click Mark Done each day to build your streak
4. Track your progress toward 30 days

## What I learned
- How localStorage works in JavaScript
- How to manage state with JavaScript arrays and objects
- How to dynamically update the DOM
- How to track dates and calculate streaks

## Tech Stack
- HTML, CSS, JavaScript (vanilla)
- localStorage for persistence
- Deployed on GitHub Pages
